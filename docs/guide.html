1. 依赖项


此条目包含$1$个可下载的文件。点击此处下载。

在安装SubbieCore之前，您的电脑需要额外的依赖项准备。它们都已经包含在上方提供的文件中。请将它们下载并解压至您的电脑，您应当得到如下三份可执行文件：

  • Anaconda$3$-$2022.05$-Windows-x$86$_$64$.exe
  • Git-$2.39.0.2$-$64$-bit.exe
  • python-$3.10.7$-amd$64$.exe

依次运行它们，按照提示完成安装。注意在安装Python时选择"Add Python to PATH"。

完成对以上$3$份软件的安装后，还需要额外配置Anaconda$3$的环境变量。在桌面右击"此电脑"，选择"属性"-"高级系统设置"-"环境变量"。如果缺少名为Path的用户变量，请通过"新建"创建；否则，将以下$5$个值添加到Path中(以安装在D:\Anaconda$3$为例)：

  • D:\Anaconda$3$
  • D:\Anaconda$3$\Scripts
  • D:\Anaconda$3$\Library\mingw-w$64$\bin
  • D:\Anaconda$3$\Library\usr\bin
  • D:\Anaconda$3$\Library\bin

最后，请对依赖环境进行检查。按下Win+R，打开"运行"界面，输入"cmd"并单击"确定"。在打开的命令行窗口中输入命令：

conda create -n test python==3.10.6

回车。待上一条命令执行结束后，执行命令：

activate test

此后，命令行内提示符前应出现(test)字样，表示已经成功搭建虚拟环境。这时再执行命令：

python -V

如果控制台输出"Python $3.10.6$"，说明依赖项的配置是成功的。


2. CUDA配置


开始此步骤前，请确保您的电脑安装了独立显卡。在此处可以查看PyTorch推荐的CUDA版本，只有推荐版本的CUDA才是被允许的。PyTorch的官方网站会指导您安装合适的PyTorch。安装完成后，不要忘记将您的显卡驱动程序升级至适配推荐的CUDA的版本。显卡运营商如NVIDIA的官方网站上一般会提供显卡驱动程序和CUDA的版本对照。注意，低于$1050$ Ti或显存小于$4$G的显卡是不能支持SubbieCore运行的。


3. 配置SubbieCore核心组件


此条目包含$1$个可下载的文件。点击此处下载。

按下Win+R，打开"运行"界面，输入"cmd"并单击"确定"。在打开的命令行窗口中输入命令：

git clone https://github.com/AUTOMATIC1111/stable-diffusion-webui.git

回车，即得到一份克隆的目录(项目的根目录)。然后下载并解压上方提供的文件，将它们复制到上述克隆目录下models/Stable-diffusion文件夹中。回到项目的根目录，在根目录下运行命令：

python -m pip install -r requirements.txt

运行完成后，SubbieCore就成功安装在了您的电脑中。


4. 启动SubbieCore


在项目的根目录下，按下Win+R，打开"运行"界面，输入"cmd"并单击"确定"。在打开的命令行窗口中输入命令：

python launch.py

运行成功后，控制台将会给出一个网址(通常是http://$127.0.0.1$:?/)。将此网址复制到浏览器中打开，即是SubbieCore的UI界面。

注意首次运行可能耗费较长时间。
